# Patlabor
This script will help you find a "main_arena" offset value for both 32-bit and 64-bit libc.so.6 during the CTF (e.g., heap-overflow chals) or your personal linux exploitations.

Tested up to libc version 2.29.

### Usage
```
$  python patlabor.py
```
